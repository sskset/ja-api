# JA API

## Prerequisites
> Have Docker installed (optional)  https://www.docker.com/   
> Have Visual Studio 2019 installed https://visualstudio.microsoft.com/downloads/   

## Startup
Set JA root project as the startup project
Set Multiple Project start
